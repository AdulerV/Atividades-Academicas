O marcador <video>

Utilizamos o marcador <video> para que o úsuario possua acesso a algum filme, vídeo, etc. No HTML 5 é suportado os seguintes formatos: MP4, WEBM, OGG.

Abaixo será citado os complementos utilizados por essa tag:

  • height: usado para definir a altura do player do video;
  • width: usado para definir a largura do player do video;
  • controls: ativa os controles de reprodução do vídeo;
  • preload: ativa o vídeo antes mesmo do úsuario ativar o player do vídeo;
  • src: define o link do vídeo;
  • loop: o vídeo será reproduzido novamente, após chegar no final;
  • muted: retira o som do vídeo;
  • poster: define uma imagem inicial, antes do úsuario clicar no player do vídeo;
  • autoplay: define que o vídeo deve ser reproduzido após ser carregado.
