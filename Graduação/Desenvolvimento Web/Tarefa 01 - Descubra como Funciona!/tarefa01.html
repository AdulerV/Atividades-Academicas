Hyper Text Markup Language

Títulos

Os títulos em HTML compõem uma lista de marcadores clássicos, isto é, presentes desde o ínicio da linguagem, no período em que ela não apresentava uma folha de estilo independente.
Sendo assim, eles representam títulos dentro de uma página HTML, sendo uma sigla para "header tag", possuindo diferentes níveis (h1, h2, h3, ..., h6).

Listas

As listas são marcadores HTML responsáveis por denotar uma lista elementos, podendo ser ordenadas (ol) ou não ordenadas (ul). Portanto, são compostas também por tags de items (li) as quais representam os elementos da lista.

Marcadores Conhecidos:

  • p
  • div
  • span
  • h1, h2, h3, ..., h6
  • ol, ul e li
  • br
  • img
  • iframe
  • video
  • audio
  • input
  • option e select
  • header
  • nav
  • section
  • article
  • footer
